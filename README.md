# online-banking-sim

A simple online banking simulator with:

1) An accounts system with the ability to create, manage, transfer between and delete accounts
2) Withdrawals and deposits
3) Direct debits
4) Interest
5) A loan system

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/online-banking-sim.git
cd online-banking-sim
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
